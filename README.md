🚀 High-Efficiency SEPIC Converter (12V to 0-30V)This repository contains the design, simulation, and hardware implementation of a SEPIC (Single-Ended Primary-Inductor Converter). Built as a part of my engineering project, it features an interactive web-based design tool and an Arduino-based 10kHz PWM controller.🛠 Component Selection & SpecificationsChoosing the right components is critical for a SEPIC converter because it must handle both input and output currents. Below is the detailed breakdown of the components used:1. Switching Element: IRLZ34N MOSFETVoltage Rating ($V_{DS}$): $55V$ (Handles our $30V$ output + $12V$ input spikes comfortably).Current Rating ($I_D$): $30A$ (Excessive for our $0.77A$ load, ensuring very low $R_{DS(on)}$ and minimal heating).Logic Level: Since I'm using an Arduino Uno, this MOSFET is ideal as it can be fully turned on with $5V$ gate logic.2. Power Diode: MBR6045WT (Schottky)Current Rating ($I_F$): $60A$ (Very robust, ensures high reliability under maximum load).Reverse Voltage ($V_{RRM}$): $45V$. (In SEPIC, the diode sees $V_{in} + V_{out}$. At $12V+30V = 42V$, this diode is working near its limit but stays safe).Forward Drop ($V_F$): Low voltage drop ($\sim 0.5V - 0.6V$), which significantly boosts efficiency.3. Inductors (Custom Wound Ferrite Core)$L_1$ (Input Inductor): $70\mu H$ (Limits input current ripple).$L_2$ (Output Inductor): $30\mu H$ (Chosen based on available ferrite core saturation).Note: Ferrite cores were used to minimize core losses at high frequencies ($10kHz$).4. CapacitorsCoupling Cap ($C_2$): $10\mu F$ Ceramic. (Crucial for energy transfer between stages).Filter Cap ($C_3$): $220\mu F$ Electrolytic. (Smooths the output voltage for sensitive loads).🧮 Interactive Design CalculatorI have developed an advanced web-based tool hosted via GitHub Pages to calculate:Duty Cycle Mapping: Based on Arduino's 10-bit PWM.Power Analysis: Input Power vs Output Power and Efficiency.Component Stress: Real-time peak current and voltage stress on the diode and MOSFET.Live Code Generation: Automatically generates Arduino C++ code based on your frequency settings.🔗 Launch SEPIC Calculator💻 Arduino PWM Control (10kHz)To minimize inductor size and audible noise, I used Timer 1 of the Arduino Uno to generate a Fast PWM at $10kHz$.Frequency: $10kHz$ (Configured via ICR1 register).Control: Potentiometer-based Duty Cycle adjustment ($0\% - 72\%$).Resolution: Restricted to $72\%$ to prevent inductor saturation and overvoltage.C++// Fast PWM Mode 14 on Timer 1
